Route Groups Explained

(public)

No auth required. Simple navbar + footer layout. Visitors can browse timetable, courses, instructors, news.

(auth)

Login & register pages. Centered card layout. Redirects to dashboard if already logged in.

(dashboard)

Auth required. Sidebar navigation. Role-based access — pages show/hide based on user role.
